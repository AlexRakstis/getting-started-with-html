How the Interweb Interconnects

The the world wide web is a lot of interconnected computers. When a you click a link such as www.imgur.com, your computer sends an HTTP Request to a server in a datacenter somewhere in serverland. The server locates the requested HTML document and transmits it back to the your computer where your web browser interprets the page and displays it on your monitor for your viewing plaesure.

Why Computers Are Instriniscally Stupid

Computers are only a smart as the programmer that programs them. They translate input in a very literal fashion. This means that a small input error by an the person coding can cause a progrom to completely fail.
